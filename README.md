# Panopticon Frontend

> 모든 서비스의 로그를 한눈에 관찰하다

Panopticon은 분산 시스템의 트레이싱, 메트릭, 로그를 통합 관리하는 **APM(Application Performance Monitoring) 대시보드**입니다.

![Panopticon Landing](./docs/imgs/사진_랜딩.png)
