# Getting Started

Panopticon 프론트엔드를 시작하는 방법입니다.

## 의존성 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

## 프로덕션 빌드

```bash
npm run build
```

## 프로덕션 서버 실행

```bash
npm start
```

## Docker로 실행하기

```bash
docker-compose up -d
```

자세한 Docker 설정은 [Docker Guide](./DOCKER_GUIDE.md)를 참조하세요.
