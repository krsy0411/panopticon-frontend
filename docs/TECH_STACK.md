# Tech Stack

Panopticon에서 사용하는 기술 스택입니다.

| -------------------- | ----------------------- |
| **Framework**        | Next.js 16 (App Router) |

| **State Management** | Zustand, TanStack Query |
| **Styling**          | Tailwind CSS 4          |
| **Charts**           | ECharts                 |
| **Animation**        | Framer Motion           |
| **Real-time**        | Socket.io               |
